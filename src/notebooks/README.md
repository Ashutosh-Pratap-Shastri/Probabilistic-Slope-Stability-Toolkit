# Probabilistic Slope Stability Toolkit

Python toolkit for reliability-based slope stability assessment using probabilistic methods.

## Research Motivation

Developed from research conducted during my M.Tech thesis:

**Stability Assessment of Critical Rock Slopes Utilizing Advanced Probabilistic Methods**

Indian Institute of Technology Patna

---

## Features

- Infinite Slope Stability Analysis
- Monte Carlo Simulation
- Reliability Index Estimation
- Probability of Failure Assessment
- Sensitivity Analysis

---

## Methodology

Input Parameters
        ↓
Random Sampling
        ↓
Factor of Safety Calculation
        ↓
Monte Carlo Simulation
        ↓
Probability of Failure
        ↓
Reliability Assessment
